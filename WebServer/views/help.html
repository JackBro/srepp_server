Uninstall

To uninstall the agent, you must have Administrator privileges. To install follow these steps depending on your operating system:
  • Windows Vista and Windows 7
    1. Click the Start Menu and select "Control Panel".
    2. In the Control Panel window, click on "Add or Remove Programs".
    3. Right-click "Summit Route EPP" from the list and select "Uninstall".
  • Windows 8
    1. From the Start Screen, click the Desktop tile. The Desktop view will open.
    2. From the Desktop, hover in the lower right-hand corner to access the Charms.
    3. Select Control Panel from the Settings charm. The Control Panel window will open.
    4. In the Control Panel window, click the "Uninstall a program" link under the Programs section.
    5. Right-click "Summit Route EPP" from the list and select "Uninstall".
